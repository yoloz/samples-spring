# Getting Started

### Spring Security OAuth

**End of Life Notice**

The Spring Security OAuth project has reached end of life and is no longer actively maintained by VMware, Inc.

This project has been replaced by the OAuth2 support provided by [Spring Security](https://docs.spring.io/spring-security/reference/servlet/oauth2/index.html) and [Spring Authorization Server](https://spring.io/projects/spring-authorization-server).
